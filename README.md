# 🩺 Medical RAG Assistant

A Retrieval-Augmented Generation (RAG) chatbot built using Streamlit and HuggingFace.

## 🚀 Features
- PDF-based question answering (CPR Guidelines)
- FAISS vector search
- Context retrieval visualization
- Evaluation module for accuracy testing

## 🛠 Tech Stack
- Python
- Streamlit
- FAISS
- HuggingFace
- LangChain

## ▶ Run Locally
```bash
streamlit run app.py
